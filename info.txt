На готовую верстке делаем:
- реализуем корзину
- в неё будут добавлться товары
- сами товары будут подгружаться сервера
- реализуем отправку данных из корзины вместе с именем и телефоном на сервер
- реализуем каталоги из шапки сайта

2:
- функционал для получения данны с сервера и выведем их на страницу
- реализация фильтрации по каталогам (из шапки профиля)

3:
- формирование корзины, с помощью из JS
- получение необходимых элементов для нее
- и работа с элементами в корзине (добавить или убрать товар)
- добавлен счётчик возле корзины(количество товаров в корзине)
- добавлена кнопка в корзине для очищения корзины
- стирает данные из корзины по кнопке 'clear cart' (недоработано)

ВОПРОС: когда мы находимся на одной вкладке и добавляем товары по очереди. 
Как только мы нажмём ещё раз на товар, которые мы уже добавили в корзину 
общее количество товаров увеличивается не на 1, а на 2. а дальше если мы 
будем дбавлять товары, которые уже есть в корзине, то обще количесвто будет 
увеличиваться, как и надо , на 1.

4:
-работа с сервером
-отправка данных

Как работать с опен сервером: 
- для этого сначала перенесем проект в опен сервер в папу domains
- и этот проект открываем в VS Code
- далее запускаем панель опен сервер(exe файл)
- внизу справа, где время, появляется флоажок опен сервера(красный), нажимаем - запустить
- если вдргу порты занята, то надо увеличить порт на один, например был 80, делаем 81
- там же где время на флажке запускаем наш проект
- можно скопировать адрес и перенести в удобный браузер
